一個用於優化 WordPress 網站的外掛，提供多種功能以提升網站效能與安全性。

== Description ==

WDO Plugin 提供以下功能：

* 移除不必要的 meta 標籤
* 停用自動產生的 CSS
* 移除不常用的腳本與樣式
* 提供自訂設定選項

== Installation ==

1. 下載並解壓縮外掛檔案。
2. 將資料夾上傳至 `/wp-content/plugins/` 目錄。
3. 在 WordPress 後台啟用外掛。
4. 前往設定頁面進行相關設定。

== Frequently Asked Questions ==

是的，WDO Plugin 已通過最新版本的 WordPress 測試。
